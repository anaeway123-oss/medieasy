## ✨ 소개

MediEasy는 병원에서 사용하는 어려운 의학용어를
일반인도 이해할 수 있는 쉬운 말로 설명해주는 서비스입니다.

사용자는

- 🔍 의학용어 검색
- 🤖 쉬운 말 번역
- 🩺 검사 결과 이해
- 📚 의학용어 학습

을 하나의 사이트에서 이용할 수 있습니다.

---

## 🚀 주요 기능

### 🔍 의학용어 검색
- 의학용어를 쉽게 설명
- 예시 문장 제공
- 관련 용어 제공

### 🤖 쉬운 말 번역
의사가 사용하는 어려운 표현을
일상 언어로 쉽게 번역합니다.

### 📚 의학용어 학습
Prefix / Root / Suffix 기반 학습

### 🎮 퀴즈
재미있게 의학용어를 익힐 수 있습니다.
